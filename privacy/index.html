Privacy Policy | Docs Online Viewer Browser Extension

Docs Online Viewer browser extension for Chrome and Firefox does not collect any personally identifiable information. No user information is collected by the extension by default. If any information will be collected, it will be done so only with explicit user consent explaining what is being collected and how it will be used.

This extension works by inserting links to external sites like https://docs.google.com beside the document links in the webpage the user visits. This process itself does not send any data to the external sites but clicking on the inserted link will expose the URL to the external website. Currently, https://docs.google.com is used as the external service to which the URLs are linked. Please refer to the https://docs.google.com/viewer privacy policy to know how it handles the data.

Docs Online Viewer browser extension may open links on the product website http://dov.parishod.com/ in order to show documentation and other helpful information. This website collects anonymous statistical data using Google analytics to better understand user interests and ways to improve the products.
